# Aperture

Voice and visual interface for GLaDOS AI assistant.

- 🎤 Speech-to-text via ElevenLabs
- 🔊 Text-to-speech via ElevenLabs (custom GLaDOS voice)
- 🌐 WebSocket-based real-time communication
- 💫 Animated visual representation with lipsync
- 🔗 Connects to OpenClaw gateway (http://192.168.50.221:18789)

## Stack

- **Backend:** Node.js + WebSocket server
- **Frontend:** Web app (browser-based, mic + speakers + display)
- **AI:** OpenClaw gateway (OpenAI-compatible API)
- **Voice:** ElevenLabs STT + TTS
